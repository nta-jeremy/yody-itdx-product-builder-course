> [!question]
> **Câu 1:** LLM là gì theo nghĩa kỹ thuật?
> - A. Bộ dự đoán token tiếp theo dựa trên context
> - B. Cơ sở dữ liệu tri thức lớn
> - C. Bộ phân loại ảnh
> - D. Hệ thống tìm kiếm ngữ nghĩa
> **Đáp án: A**
> **Giải thích:** LLM (Large Language Model) là mô hình sinh token — mỗi bước dự đoán token tiếp theo có xác suất cao nhất dựa trên chuỗi token đầu vào.

> [!question]
> **Câu 2:** Khi nào nên dùng RAG thay vì chỉ prompt thuần?
> - A. Khi cần deterministic output
> - B. Khi cần truy cập knowledge base ngoài training data
> - C. Khi cần tăng temperature
> - D. Khi cần giảm context length
> **Đáp án: B**
> **Giải thích:** RAG (Retrieval-Augmented Generation) cho phép model lấy thông tin từ knowledge base cập nhật liên tục mà không cần fine-tune lại model.

> [!question]
> **Câu 3:** Temperature = 0 có nghĩa là gì?
> - A. Model không hoạt động
> - B. Model chọn token có xác suất cao nhất (greedy)
> - C. Model sinh output ngẫu nhiên hoàn toàn
> - D. Model dùng context dài hơn
> **Đáp án: B**
> **Giải thích:** Temperature = 0 → greedy decoding → model luôn chọn token có xác suất cao nhất → output deterministic.
